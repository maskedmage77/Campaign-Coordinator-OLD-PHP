Spells Help


To view more information about a spell click on it!

Click the Create Spell button to add a new spell to the list.

The Create Spell menu supports markup for bolding and italics. This will only work in spell descriptions.
  • To bold add two astrisks on both sides
    • **Example** = Example
  • To italicize add a single astricks on both sides
    • *Example* = Example
  • These can be combined
    • ***Example*** = Example
To select multiple classes for a spell you need to hold the ctrl key and click them. This may not be applicable on mobile and will vary by platform.
